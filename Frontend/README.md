# Doc2Code

Turn any API documentation into production-ready code instantly. Paste a documentation URL, describe what you need, and get clean code in TypeScript, Python, Java, Go, Rust, or JavaScript.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at [http://localhost:8080](http://localhost:8080).

## Tech Stack

- **React 18** + TypeScript
- **Vite** – build tool
- **Tailwind CSS** – styling
- **shadcn/ui** – component library
- **Zustand** – state management
- **React Router** – routing
- **Framer Motion** – animations
- **React Syntax Highlighter** – code display

## Scripts

| Command | Description |
